# Reading Logs In Grafana

Grafana is a web application that makes logs and metrics much easier to explore.

## Beginner workflow

1. Start the observability stack in `infra/observability`.
2. Start the Spring Boot application.
3. Open Grafana.
4. Search logs for an employee identifier such as `EMP-1001`.
5. Compare the request logs with the payroll response logs.

This is a gentle way to see how requests move through the system.
