# Repository Tour

Use this page like a map when you are learning the project.

## Important files

- `pom.xml`: root Maven build and shared quality properties.
- `spring-control-client/pom.xml`: Java application dependencies and quality
  plugins.
- `cobol-core/src/main/cobol/payroll_calculation_engine.cbl`: the business
  calculation engine.
- `.github/workflows/quality.yml`: linting, tests, coverage, and repository
  policy checks.
- `.github/workflows/security.yml`: dependency, secret, static, and supply-chain
  checks.
- `.github/workflows/documentation.yml`: GitHub Pages build and deployment.
